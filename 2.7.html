1.7 Filetypes

When adding an image to a website it is important to choose a suitable file type as an unsuitable file type can result in a loss of image quality or slow down the speed of the webpage. Photographs are best suited to a JPEG format as it allows the file size to be reduced without heavily affecting page speed or the photoâ€™s quality. PNG format is great for logos as it allows for a high level of image detail and keeps the graphic at a high quality, though some web browsers do not support the PNG format so that should be kept in mind when choosing your image file types.
